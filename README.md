# vacation-registration-app
Vacation Registration App is an Android application designed to help users manage and track their vacations in an easy and organized way.  The app allows users to add vacation records, view history, and manage holidays efficiently.  It is built to simplify vacation management for both personal and professional use.

## 📖 About The Project

**Vacation Registration App** is a professional Android application that helps employees and HR managers manage, track, and organize vacation requests efficiently.

> 📌 **Naming Clarification:**
> - **Google Play Name:** تسجيل الاجازات  
> - **Package Name:** `com.shriefaboulailah.enjoyVacation`  
> - **Repository Name:** `vacation-registration-app`  
>  
> All refer to the same application.

---

## ✨ Key Features

- ➕ Add new vacation requests (Casual, Sick, Annual, etc.)
- 📋 View full vacation history
- 📊 Track official holidays in Egypt
- 🎨 Clean and intuitive UI (Material Design)
- ⚡ Optimized for performance and smooth user experience
- 📱 Easy navigation and responsive layout
